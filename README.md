# CLI ToDoリストアプリケーション (Java)

## 概要 (Overview)

本アプリケーションは、Javaの**コマンドラインインターフェース (CLI)**で動作するシンプルな ToDo リスト管理ツールです。

ファイルI/O (`java.io` または `java.nio`) とコレクション (`ArrayList` など) を使用し、ToDo項目の永続化（データの保存・読み込み）と効率的なデータ操作を実装することで、Javaの基礎力を確認することを目的としています。

## 主な機能 (Features)

1.  **項目の追加**: ToDoの内容、期限などをリストに追加。
2.  **項目の表示**: 現在のToDoリスト全体を表示。
3.  **項目の完了**: 特定の項目を完了としてマーク。
4.  **リストの保存**: リストの内容をファイルに書き込み。
5.  **リストの読み込み**: アプリケーション起動時にファイルからリストを復元。
